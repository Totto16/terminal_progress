# Terminal Progress

## What is this

This implements the OSC Terminal escapes for progress bars, from [ConEmu](https://conemu.github.io/en/AnsiEscapeCodes.html#ConEmu_specific_OSC).

Why? Becasue Ghostty also supports those, and I use ghostty.
